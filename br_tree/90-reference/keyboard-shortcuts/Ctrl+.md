The CTRL key in combination with another key can be used in several different ways, including:

*`Ctrl+Right Bracket`
*`Ctrl+A` for entering `ATTN Mode`
*`Ctrl+Tab`
*`Ctrl+Shift+Tab`
*`Ctrl+Y`
