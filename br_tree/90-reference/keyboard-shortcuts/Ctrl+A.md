**Ctrl+A** is the default key used to get `Attn` during a running BR! `program`.

:Note: If you want to remap **Ctrl+A** to something else using 'config `keyboard`', the hex code for **Ctrl+A** is 01. See the `Keyboard` `brconfig.sys` specification for more information.
