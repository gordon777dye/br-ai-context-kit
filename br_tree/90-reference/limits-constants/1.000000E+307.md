The number **1.000000E+307** is the largest number that can be used in `Business Rules!`.  See also `System Limits`. This however may varies according to your hardware (CPU). The system function Inf (e.g., type PRINT INF) returns the largest number. 1.000000E+307 is the largest number on all DOS, Unix and Linux versions at the time of publication of this manual.
