**Error:** 4334

**Description:** Novell error 134 or Unix / Linux error 34

**Details:** As of BR! version `4.16` This error no longer exists.  It was replaced by a combination of Error Number 4320 used in conjunction with the system variables SYSERR and SYSERR$.Novell: Create file exists read only. A station has attempted to create a file with the same name as a file that already exists under read-only status.Unix / Linux: Result too large. Possibly due to corrupted memory. Try reloading from source.
