**Error:** 4224

**Description:** As of 4.3: File is too large for locking mode. Before 4.3: `Operating System Error` 24 (Bad request structure length)

**Details:** - This error would normally occur in `Business Rules!` only if memory has been corrupted. Bad request structure length.
- The size of the file exceeds the record locking limitation #Try reloading from `source`. If this doesn't work, contact your dealer or `BRC` for assistance.
- increase the `Maximum#Record_Locking record locking limitation`
