**Error:** 0807

**Description:** Invalid `border` specification in `open` `statement` for a `window`. As of 4.30, border characters can no longer be specified.

**Details:** The border spec was changed and some of the letters that we used to use are no longer valid, I believe only D and S are valid, the others cause an error.
