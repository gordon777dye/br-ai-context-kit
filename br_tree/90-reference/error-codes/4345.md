**Error:** 4345

**Description:** Novell error 145

**Details:** As of BR! version `4.16` This error no longer exists.  It was replaced by a combination of Error Number 4320 used in conjunction with the system variables SYSERR and SYSERR$.Rename partially fails (other names already exist). A station has attempted to rename files using as ambiguous file name. The operation was only partly successful because one or more files matching the ambiguous specifications already exists.
