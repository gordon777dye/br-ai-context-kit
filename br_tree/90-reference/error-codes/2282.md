**Error:** 2282

**Description:** `Library Functions

**Details:** Library Function` not processed by `Library` `statement` A function call has been issued for a function that has not been processed by a Library statement. Correct the function call, name the function in a Library statement, and / or make sure that the Library statement that references the function is being executed prior to the function call.
