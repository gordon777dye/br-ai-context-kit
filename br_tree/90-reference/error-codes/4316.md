**Error:** 4316

**Description:** Unix / Linux error 16

**Details:** As of BR! version `4.16` This error no longer exists.  It was replaced by a combination of Error Number 4300 used in conjunction with the system variables SYSERR and SYSERR$.Device or resource busy. An attempt was made to mount a device that was already mounted or attempt was made to dismount a device on which there was no active file. The device or resource is currently unavailable
