**Error:** 0727

**Description:** DEPRECATED ERROR (as of 4.3) FORM Variable not referenced

**Details:** A string variable referenced a USING clause does not appear anywhere else in the program. Change the variable in either the FORM or USING statements, or change the program; they must match.
