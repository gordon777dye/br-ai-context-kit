**Error:** 1125

**Description:** Invalid `multi-lined If` operator

**Details:** Multi-lined IF operator not preceded by a multi-lined If.  Or `For`/`Next` loop conflict with multi-lined if. Make sure that the unmatched `End If` or `Else` statement is preceded by a multi-lined If `statement`.
