**Error:** 4338

**Description:** Novell error 138 or Linux error 38

**Details:** As of BR! version `4.16` This error no longer exists.  It was replaced by a combination of Error Number 4320 used in conjunction with the system variables SYSERR and SYSERR$.Novell: No delete privileges. A station has attempted to delete a file from a directory for which it has no delete privileges.Linux: Not available
