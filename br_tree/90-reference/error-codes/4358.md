**Error:** 4358

**Description:** Novell error 158

**Details:** As of BR! version `4.16` This error no longer exists.  It was replaced by a combination of Error Number 4320 used in conjunction with the system variables SYSERR and SYSERR$.Invalid file name. Attempt to create a file using invalid characters within the name of the file or attempt to open a file that does not exist
