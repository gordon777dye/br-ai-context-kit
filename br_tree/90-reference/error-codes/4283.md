**Error:** 4283

**Description:** Operating System Error 83 (Error in updating previous/next link)

**Details:** - A file error occurred while system was updating the previous or next link. The file is probably corrupted.
- Fail on INT 24 due to hardware problem or corrupted memory. This error is reported to Business Rules by the operating system; see your Operating System manuals for more information. Rebuild the list or recover data.
