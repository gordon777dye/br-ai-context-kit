**Error:** 0632

**Description:** Problem with `Index File`

**Details:** - Wrong version of the index file for `Collate Alternate`. As of BR version `4.17`q, any revisions are done at the time the keys are built rather than trying to perform collating adjustments while comparing keys.  
- Corrupted Index file or Index file contains duplicate `keys` and index is not supposed to contain duplicate keys. #Re-index the File.
- Delete the duplicated keyed records and `reindex` the file or rebuild the index with `DUPRECS` to allow duplicate records.
