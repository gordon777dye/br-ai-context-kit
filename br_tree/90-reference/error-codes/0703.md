**Error:** 0703

**Description:** Illegal access by <nowiki>REC=</nowiki>

**Details:** - Attempt to access a sequential file with a <nowiki>REC=</nowiki> clause.
- Attempt to access an internal file with a <nowiki>POS=</nowiki> clause. # Open the file `RELATIVE` or remove the <nowiki>REC=</nowiki> clause.
- Open the file `EXTERNAL` or remove the <nowiki>POS=</nowiki> clause.
