**Error:** 4360

**Description:** Novell error 160

**Details:** As of BR! version `4.16` This error no longer exists.  It was replaced by a combination of Error Number 4320 used in conjunction with the system variables SYSERR and SYSERR$.Subdirectory not empty. Attempt to delete a subdirectory that still contains files in it.
