**Error:** 7603

**Description:** Duplicate key found

**Details:** The index is complete; duplicate keys were found. The index was created. The application program indicated that duplicate keys were not to be allowed. Enter GO if running a procedure or enter CONTINUE if the index was issued by a program. Program changes may be necessary to delete the duplicate keys and to prevent others from being added.
