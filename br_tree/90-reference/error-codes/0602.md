**Error:** 0602

**Description:** Key file name conflict

**Details:** Key file name is not an index file, or the specified key file name and master file name are the same file.This error may also occur when S/23-MODE is ON and an OPEN statement tries to open a key file as an internal file. Change the key file name.If the S/23-MODE is ON, correct the NAME= parameter in the OPEN statement so that it refers to a display file.
