**Error:** 2270

**Description:** `Library` not found

**Details:** - If `Clear` was used, this error means the specified library doesn't exist in memory.
- The specified library was not found on disk. (A library statement's path may be misspelled) #If CLEAR was used make sure the library is loaded when CLEAR is executed. 
- Check the file name for errors and Verify the file's disk location.
