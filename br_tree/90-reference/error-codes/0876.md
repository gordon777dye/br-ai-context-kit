**Error:** 0876

**Description:** Use of a `grid` only specifications when processing a `ListView`

**Details:** - Attempt to read a single cell instead of a row on a ListView.
- statement encountered in non-GUI mode #Using grid only syntax when accessing a ListView.
- Perhaps add an "`if` `env$`('`GUIMode`')<nowiki>=</nowiki>"ON" then' to the beginning of the statement.
