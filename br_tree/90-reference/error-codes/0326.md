**Error:** 0326

**Description:** A literal is passed as a variable to a function that wants to return a value.

**Details:** A literal is passed as a variable to a function that wants to return a value.You attempted to assign a value to a literal. You can't. Put the value into a variable and pass the variable instead.Do not expect the variable to have the same value when it has returned from the function call
