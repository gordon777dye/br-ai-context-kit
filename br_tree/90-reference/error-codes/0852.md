**Error:** 0852

**Description:** More `Variables` than `fields`

**Details:** - The number of populated I/O variables exceeds the number of fields in `PRINT FIELDS`, `INPUT FIELDS`, `RINPUT FIELDS`, `INPUT SELECT` or `RINPUT SELECT`.
- Print Fields with empty form statements #Increase the number of elements in the field definition or decrease the number of I/O variables.  Also make sure all field definition variables do not contain blanks values.
- put some values in those form statements
