**Error:** 4326

**Description:** Unix / Linux error 26

**Details:** As of BR! version `4.16` This error no longer exists.  It was replaced by a combination of Error Number `4320` used in conjunction with the system variables `SYSERR` and `SYSERR$`.Text files busy. File is being used by another workstation and can't be opened for writing
