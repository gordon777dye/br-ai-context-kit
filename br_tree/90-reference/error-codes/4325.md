**Error:** 4325

**Description:** `Unix` / `Linux` error 25

**Details:** As of BR! version `4.16` This error no longer exists.  It was replaced by a combination of Error Number `4320` used in conjunction with the system variables `SYSERR` and `SYSERR$`. Not a character device. Request can be processed only for special devices, such as diskette drives
