**Error:** 0053

**Description:** Attempt to `write` to an existing `record`

**Details:** A write `statement` used the keyword `REC` for an existing record rather than for a deleted record; writing to an existing record is not allowed. Correct the WRITE statement that caused the error. To specify a record that has not been deleted, remove the <nowiki>REC=</nowiki> clause and write to the end of the file or use `REWRITE` to overwrite the existing record.
