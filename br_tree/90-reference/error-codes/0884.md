**Error:** 0884

**Description:** Invalid backward reference to help field

**Details:** Invalid backward reference to help field. The "&field-number" parameter in a help string is invalid. Change the "field-number" parameter so that it refers to a field that comes before the current field. Make sure that it is followed by a semi-colon (;), even if a different separator is used for the text in a help string.
