**Error:** 7805

**Description:** Syntax error in record specification

**Details:** A syntax error occurred in the RECORD specification. Run PRESORT and check the RECORD specification in the sort control file for typing mistakes, missing punctuation, etc. If PRESORT is not used, check the punctuation in the program that creates the sort control file.
