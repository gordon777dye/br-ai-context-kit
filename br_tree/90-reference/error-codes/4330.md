**Error:** 4330

**Description:** Novell error 130 or Unix / Linux error 30

**Details:** As of BR! version `4.16` This error no longer exists.  It was replaced by a combination of Error Number 4320 used in conjunction with the system variables SYSERR and SYSERR$.Novell: No open privileges. A station without open privileges has attempted to open a file.Unix / LinuxRead-only file system. Attempt to modify a file or directory on read-only mounted device.
