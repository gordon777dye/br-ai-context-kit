**Error:** 7611

**Description:** Index file already exists

**Details:** Specification of an index file that already exists and that data is not to be destroyed. This error also occurs when the specified file is not internal or if the name is not valid. Use a different file name to create a new index file, use a file that is internal, or specify the REPLACE keyword
