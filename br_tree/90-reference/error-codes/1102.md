**Error:** 1102

**Description:** END IF or END DEF followed by another statement

**Details:** An END IF or END DEF statement has been followed by a colon and another statement. The only statement, which may legally follow END IF or END DEF, is the END statement. Remove the statement following the END IF or END DEF and place it on another line.
