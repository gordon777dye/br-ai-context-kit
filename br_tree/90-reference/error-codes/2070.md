**Error:** 2070

**Description:** DEPRECATED ERROR as of 4.30: Not enough memory

**Details:** A `Load` or `Save` operation was terminated because of insufficient memory. LIST the program to a data file (via LIST  > temp.brs) and recover it before proceeding.For example
LIST  >program.brs
LOAD program.brs,SOURCE
REPLACE program.br
