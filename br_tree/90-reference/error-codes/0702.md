**Error:** 0702

**Description:** File not open `Keyed` or `Linked`

**Details:** The I/O statement attempt to access a file in a manner that does not match the file open. Some possibilities:
- Attempt to read a non-keyed file with a `Key` or `Search` parameter
- Attempt to read non-linked file with <nowiki>Link=</nowiki> parameter. Change the `Read` statement or the `Open` statement to allow access as desired.  For situations described above:
- Remove the Key or Search clause from the error-causing program statement, or open the file as Keyed.
- Remove the <nowiki>Link=</nowiki> parameter from the error-causing statement or open the file as Linked.
