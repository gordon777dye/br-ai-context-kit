**Error:** 1022

**Description:** Missing keyword

**Details:** A keyword is missing. For example, the following statement is missing the "THEN" keyword:1000 IF X=6 GOTO 1200The correct statement would be:1000 IF X=6 THEN GOTO 1200 Add the missing keyword.
