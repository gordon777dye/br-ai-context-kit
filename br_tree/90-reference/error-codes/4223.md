**Error:** 4223

**Description:** DEPRECATED ERROR, as of 4.30: Operating System Error 23

**Details:** Data error (CRC)Disk or diskette sector is bad. Enter GO to retry the operation. If unsuccessful, reformat the media. If it's a diskette, just get a new one.
