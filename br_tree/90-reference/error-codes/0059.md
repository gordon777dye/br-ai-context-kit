**Error:** 0059

**Description:** Attempt to rewrite over key field — **DEPRECATED** (2.x releases only)

**Details:** In 2.x releases a REWRITE that changed the key field in the master record raised this error. **This error is deprecated:** current BR permits a REWRITE to change the key field value, and automatically updates all related key files opened as a group.
