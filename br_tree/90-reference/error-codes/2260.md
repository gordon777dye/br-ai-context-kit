**Error:** 2260

**Description:** Data space exceeds 1MB

**Details:** Data space exceeds 1MB in length. If changes have been made to the `program`, list it to a file or `save` it in source for safekeeping. Reduce the number of `variable` values by reducing `array` elements and `retry` the program.
