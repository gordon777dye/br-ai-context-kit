**Error:** 2278

**Description:** Retain release conflict

**Details:** One of the following situations has occurred:
- A `library` containing an `OPTION` RETAIN statement has been referenced by a LIBRARY statement with the `RELEASE` keyword.
- A library has been referenced with or without RELEASE. # Revoke the OPTION RETAIN / RELEASE conflict.
- Make sure that all references to the library are handled consistently regarding RELEASE.
