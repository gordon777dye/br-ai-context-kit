**Error:** 2111

**Description:** Missing lines for a program in object code

**Details:** Lines for an object program are missing. For example, a program with a GOTO statement that references a line, which does not exist, will cause this error when the following command is issued:SAVE program OBJECT Resolve line reference problems with lines added to an object (.WBO) program before attempting to SAVE or REPLACE a program.
