**Error:** 0904

**Description:** Illegal Character used in an `BR! Environmental Variable`

**Details:** You tried to use a space, a period, space or another invalid character in a `SetEnv` command as the variable name. Use a valid variable name, that does not include spaces, periods, or other invalid characters.
