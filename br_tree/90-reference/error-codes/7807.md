**Error:** 7807

**Description:** Syntax error in MASK specification

**Details:** A syntax error occurred in the MASK specification. Run PRESORT and check the MASK specification in the sort control file for typing errors, missing punctuation, etc. If PRESORT is not used, check the punctuation in the program that creates the sort control file.
