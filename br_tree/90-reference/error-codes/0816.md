**Error:** 0816

**Description:** Illegal `Form` specification

**Details:** Use of an illegal format specification in Form `statement`. Change the Form statement so that it only uses `format specifications`, which are valid for the I/O statement you want. For e.g.: you cannot specify the `PD` format spec with a `Print` statement.
