**Error:** 0877

**Description:** `GUI` On only `statement` encountered in GUI Off mode

**Details:** GUI On only statement encountered in GUI Off mode.
`Option (config)` 41 prevents this error. OPTION 41 suppresses this error and ignores GUI syntax when ENV$("`GUIMode`")<nowiki>=</nowiki>"OFF"
