**Error:** 4349

**Description:** Novell error 149

**Details:** As of BR! version `4.16` This error no longer exists.  It was replaced by a combination of Error Number 4320 used in conjunction with the system variables SYSERR and SYSERR$.File detached. A station has attempted to read or write to a file that has been detached.
