**Error:** 2083

**Description:** Array re-dimensioned too big

**Details:** You tried to re-dimension an array larger than 1MB. No single array may exceed 1MB. Try to break your array up into smaller parallel arrays. Reduce the number of elements or the length of each element in the array.
