**Error:** 4313

**Description:** Unix / Linux error 13

**Details:** As of BR! version `4.16` This error no longer exists.  It was replaced by a combination of Error Number 4300 used in conjunction with the system variables SYSERR and SYSERR$.Permission denied. File accesses attempt forbidden by protection system.
