**Error:** 1121

**Description:** Open ended THEN or ELSE not allowed

**Details:** An open-ended THEN or an open-ended ELSE is specified on an ELSE clause. This error code was implemented in BR release 3.65c to reduce ambiguity in the IF structure. Programs that Worked prior to this release will encounter an error on 3.56c+ releases after when they are recompiled from source if they utilize this programming method. Move the ELSE clause to a separate statement. Or, if this BR change causes an extended problem in your applications, contact `BRC` for a work around.
