**Error:** 0058

**Description:** I/O `string` overflow

**Details:** - An I/O statement (`Read`, `Write`, `Print` or `Input`) tried to read data into a string `variable` that wasn't `dimensioned` large enough.
- When creating a `Grid and List` # Shorten the string or increase the dimension length of the string variable.
- The FORMS$ matrix field lengths **have** to agree (DIM EXACTLY) with the variables that you are reading them into/out of.
