**Error:** 7830

**Description:** Invalid limits in RECORD

**Details:** Lower limit is greater than the upper limit in a sort RECORD specification. RECORD 0,11,1,C,"a","Z" is invalid because "a" has an ASCII value of 97 and "Z" is 90. Uppercase letters come before lowercase letters in ASCII. Run PRESORT and change the limits so that they are ordered according to the current ASCII collating sequence
