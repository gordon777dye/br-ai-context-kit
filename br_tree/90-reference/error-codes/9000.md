**Error:** 9000

**Description:** Out of memory

**Details:** Critical out of memory error. The program is in danger of being corrupted or lost. Before attempting any other actions, save your program with the LIST command (Example: LIST  >filename.BRS), then reload it from source (LOAD filename SO) and save it with the SAVE command. Exit and re-enter Business Rules before continuing.
:To prevent the error from occurring again, reduce the size of the arrays, the program or both.
