**Error:** 0301

**Description:** `Def` conflict number of parameters.

**Details:** `User defined function` parameters do not match the parameter list in the Def `statement`. Ensure that the `variable` in the user function are of the same type (`numeric`, `string`, `array`, etc) as those in the Def statement.
