**Error:** 0501

**Description:** Not enough data for INPUT

**Details:** The record read from the screen or a display file does not contain enough data for the INPUT statement. Change the input statement to read the correct number of parameters, or re-prompt the operator for the correct number of values or fix the data file. Also see the INPUT statement for information about the effects of the forward slash (/) on input.
