**Error:** 1026

**Description:** Illegal `expression`

**Details:** - Use of an illegal expression. - For example, a `variable` or `function` name that is `maximum too long`.
- `string literal` too large
- a new `array` is attempted to be created in a program at `run time` #Check string and numeric expressions for syntax errors or errors in logic.
- String literals may be no greater than 250 characters in length.
- end the program and dim the array in the program before attempting to use it.
