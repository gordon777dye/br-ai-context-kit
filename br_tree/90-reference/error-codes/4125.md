**Error:** 4125

**Description:** Data file `VERSION` error

**Details:** `VERSION` number of existing file does not match `VERSION` number specified in OPEN INTERNAL statement. Update the file if it is old or correct the `VERSION` number in the OPEN INTERNAL statement.
