**Error:** 4354

**Description:** `Novell` error 154

**Details:** As of BR! version `4.16` This error no longer exists.  It was replaced by a combination of Error Number `4320` used in conjunction with the system variables `SYSERR` and `SYSERR$`.Renaming across volumes. Attempting to `rename` a file on a volume other than the user's current volume.
