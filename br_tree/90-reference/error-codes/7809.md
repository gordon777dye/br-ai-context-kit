**Error:** 7809

**Description:** Syntax error in SUM specification

**Details:** Syntax error in SUM specification. Run PRESORT and check the SUM specification in the sort control file for typing errors, missing punctuation, etc. If PRESORT is not used, check the punctuation in the program that creates the sort control file.
