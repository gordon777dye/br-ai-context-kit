**Error:** 1006

**Description:** Syntax error

**Details:** - A syntax error has occurred. The specified combination of keywords and expression is self-conflicting
- A `config` `attribute` `statement` that specified an attribute name which was too long or invalid. #Correct the line for required punctuation and keywords, and typing errors such as misspellings. 
- Shorten the attribute name to be 12 characters or less.  `Attribute` names can not be more than 12 characters in length.
