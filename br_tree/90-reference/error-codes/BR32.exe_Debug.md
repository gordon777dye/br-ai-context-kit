The **Debug version of BR32.exe** is used by beta testers and developers of the Bussiness Rules! language.

**BR32.exe Debug** is a enhanced version of `BR32.exe` which will create crash dumps and  `ftp` these error reports to `BRC`.

As of `4.3` debug versions of BR now expect you to use a `Logging` `configuration statement`. This enables BR to post exit messages during unexpected terminations.
