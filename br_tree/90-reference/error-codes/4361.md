**Error:** 4361

**Description:** Novell error 161

**Details:** As of BR! version `4.16` This error no longer exists.  It was replaced by a combination of Error Number 4320 used in conjunction with the system variables SYSERR and SYSERR$.Directory I/O error. A non-recoverable I/O error has occurred on the disk in the directory area. The error has occurred in both redundant directories and is fatal.
