**Error:** 0121

**Description:** Illegal reference to an `array`

**Details:** An illegal reference to an array exists in the `program`. Change the program so that it defines and uses arrays properly. Check that your program does not try to use a two-dimensional array as a one-dimensional array or vice-versa.
