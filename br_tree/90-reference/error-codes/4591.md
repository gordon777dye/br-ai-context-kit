**Error:** 4591

**Description:** Process killed by `timeout`

**Details:** Timeout on a `Shell Call` (Windows or MacIntosh) Insert a -c in your shell callie: Execute "sys -m -c program_name.exe -optional_parameters", so BR will continue without the error if it times out.Even though the process timed out during a shell call, the process remains running (at least in Windows XP, and MacIntosh anyway)To cease this error entirly add the following to your BRCONFIG.SYS file:SHELL LIMIT -1This will prevent any timeouts.
