**Error:** 606

**Description:** Invalid element in `Open` `statement`

**Details:** The Open statement includes an invalid element in the KPS<nowiki>=</nowiki> or KLN<nowiki>=</nowiki> string. Some possibilities:
- The record length, key position or key length for an Open statement is either missing or invalid.
- `Open Linked` statement attempted to use first 8 bytes of record as key field (releases `3.60` and higher). #Add or correct the `RecL`<nowiki>=</nowiki>, KPS<nowiki>=</nowiki> and/or KLN<nowiki>=</nowiki> parameters.
- don't do that.
