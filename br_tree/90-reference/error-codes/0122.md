**Error:** 0122

**Description:** Illegal `array` element

**Details:** - An invalid array element has been specified.
- (Macintosh) Reference to mapped network drive lost
- double instance of an `or` in an `If statement` # Check to see that each specified element falls within the array's allowable range. It must not be larger than the `dimensioned` (or re-dimensioned) value, or negative, or invalid for the `BASE` in which the program is functioning.
- Mount network drive
- remove the repeated or
