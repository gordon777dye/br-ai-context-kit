**Error:** 4332

**Description:** Novell error 132 or Unix / Linux error 32

**Details:** As of BR! version `4.16` This error no longer exists.  It was replaced by a combination of Error Number 4320 used in conjunction with the system variables SYSERR and SYSERR$.Novell: No create privileges. A station without create privileges has attempted to create a file.Unix / Linux: Broken pipe. Possibly due to corrupted memory. Try reloading source.
