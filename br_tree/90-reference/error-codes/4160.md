**Error:** 4160

**Description:** Open files limit exceeded. See `System Limits`. As of 4.30: Out of internal file handles

**Details:** File open exceeded BR's internal limit on number of open files. Make sure you don't have too many files open.
BR's internal limit is 594 in the Windows Release.
You may have actually run out of OS file handles.
