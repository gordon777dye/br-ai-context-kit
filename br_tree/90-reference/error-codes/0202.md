**Error:** 0202

**Description:** Function not active

**Details:** - An `FNEND` statement was encountered when no function was active.
- A Function called a `GOSUB` and attempted to exit the function (via FNEND) before returning from the gosub #Fix the program so that the FNEND statement is encountered only when a function is active.
- Repair the faulty logic, make sure there is no way out of your gosub except by `RETURN`
