**Error:** 1040

**Description:** Illegal `numeric` `expression`

**Details:** - Use of an illegal numeric expression. Attempt to print a `variable` value from immediate mode when the variable is a valid abbreviation for a `Business Rules!` `keyword`.
- Use of a numeric variable that is too long
- `Srch` parameters are juxtapositioned.
- `keyword` (like `menu`) used as a `variable`.
Note: If you are `proc`ing in a program the error may be on the next line #Precede the name of the variable you are trying to print with the `print` `statement`. Check numeric expressions for syntax errors or errors in logic
- Use a shorter variable name
- switch the parameters within the Srch call to the correct sequence.
- Choose a different variable name
