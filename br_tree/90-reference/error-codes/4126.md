**Error:** 4126

**Description:** File abnormally truncated

**Details:** The `BR Header` does not match the actual # of `records` in the data file.
For example a Copy filed to copy the entire file or `Business Rules!` was terminated in the middle of a program, by pressing Ctrl+Alt+Delete.

Usually this means that some data was lost.
