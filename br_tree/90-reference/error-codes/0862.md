**Error:** 0862

**Description:** Missing field length

**Details:** - The format specification does not list a field length. 
- Missing ",RowSub,SelOne" from a Input List specification
- Incorrect syntax in a print fields statement, such as:

  print fields "2,2,Cr 10.[W]":"Forwarder:" #Provide the missing information
- Provide the missing information
- Correct the syntax by replacing the inappropriate period with a comma.
