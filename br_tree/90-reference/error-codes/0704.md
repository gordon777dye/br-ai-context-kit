**Error:** 0704

**Description:** `File handle` not open

**Details:** Attempt at accessing an unopened file. `Open` the file before attempting to access it or correct the file handle referenced in the `statement` that caused the error.  Perhaps adding an `IOErr` condition to the end of a `Close` statement.

See Also:  `File (internal function)`
