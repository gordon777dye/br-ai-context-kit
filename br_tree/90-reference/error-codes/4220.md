**Error:** 4220

**Description:** DEPRECATED ERROR, as of 4.30: `Operating System Error` 20 (Unknown unit)

**Details:** This error would normally occur in Business Rules only if memory has been corrupted. Unknown unit. Try reloading from source. If this doesn't work, contact your dealer or `BRC` for assistance.
