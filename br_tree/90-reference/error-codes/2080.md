**Error:** 2080

**Description:** Insufficient dimensioned storage

**Details:** Insufficient memory exists. There is not enough memory to allocate strings or arrays. Eliminate some arrays and/or make them smaller or delete program lines to make the program smaller. Also try saving the program and loading it again.
