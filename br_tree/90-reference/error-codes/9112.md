**Error:** 9112

**Description:** `REPLACE` aborted

**Details:** REPLACE aborted. The program was corrupted and the attempted REPLACE operation was not completed. Neither the output file nor the program on disk will be useable. To recover, immediately execute LIST  > Workfile command and then reload from source. Otherwise, your program on disk and in memory may be lost.
