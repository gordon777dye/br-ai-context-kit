**Error:** 0434

**Description:** `Line number` too long during `Edit ~`

**Details:** When BR! was attempting to merge changes (via Edit ~) an invalid line number was encountered.  The line number is too long (too many digits). Correct the line number in your `editor` and perform edit ~ again.
