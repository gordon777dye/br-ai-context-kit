**Error:** 2082

**Description:** #Data structure too long
#source code corruption

**Details:** - Data structure too long.
- source code corruption #Make sure `arrays` do not exceed their `System Limits#Arrays maximums`.
- Save the program as `source` and `reload` it and `replace` it.
