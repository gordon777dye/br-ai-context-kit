**Error:** 9305

**Description:** System error on user-defined function

**Details:** An error occurred while processing a `user-defined function`. It is likely that a `statement` failed to compile properly or has become corrupted. Save the `program` as `source` and reload it as source. If the error persists, call your dealer or `BRC` for help.
