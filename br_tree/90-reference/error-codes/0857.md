**Error:** 0857

**Description:** Illegal `column` specification

**Details:** Illegal column parameter the beginning or end of a field specified in `Full Screen Processing Statements` is out of range. Type `Cnt` +1 to return the subscript of the bad element in the field definition `array`. All fields must be within columns 1 and it’s `Parent Window parent's` Window's width.
