**Error:** 9992

**Description:** `Def` `statement` compiler error

**Details:** Def statement compiler error. Save the program as `source` and reload it as source. If the `Save` command does not work often `LIST ` >TEMP.BRS will work. If the error persists, call your dealer or `BRC` for help.
