**Error:** 4210

**Description:** DEPRECATED ERROR, as of 4.30: `Operating System Error` 10 (Invalid environment)

**Details:** This error would normally occur in Business Rules only if memory has been corrupted. (invalid environments) Try reloading from source. If this doesn't work, contact your dealer or `BRC` for assistance
