**Error:** 0854

**Description:** Illegal `row` specification

**Details:** - Illegal specification found for the "row" parameter in the `field` definition of a `full-screen processing` for `statement`.
- Program made in `3.91`, attempted to run in `3.67`. # Type `CNT` +1 to return the subscript of the bad element in the field definition array. Change the field definition's ROW value to a number between 1 and 24. Or (if dealing with a window) to a value that does not exceed the number of rows in the window.
- Must save as source in 3.91 (not 3.67) and load in 3.67
