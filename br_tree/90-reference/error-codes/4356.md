**Error:** 4356

**Description:** Novell error 156

**Details:** As of BR! version `4.16` This error no longer exists.  It was replaced by a combination of Error Number `4320` used in conjunction with the system variables `SYSERR` and `SYSERR$`. Invalid path. Attempt to access a path that does not exist or that contains invalid characters.
