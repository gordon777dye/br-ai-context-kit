**Error:** 6210

**Description:** Passthrough printing

**Details:** Windows Printer Driver does not support PASSTHROUGH operation
- Incorrect Printer Driver was selected
- Alternate Printer Driver Needed
- EMF Spool Data Format Use DIRECT printing (DIRECT:/printername).
- Select Quit and then run program again, selecting correct printer driver (If youset your default printer correctly you can avoid getting this error in the future.
- If you've completed the Printer Setup stepsand are still receiving this error, try using an alternate printer driver. HPDeskJet 500 (not 500c nor 500+) seemsto work well for most printers. If itdoes not, try alternate printer drivers.
- Set the printer driver's Spool Data format to RAWAdditional Notes: How to Select a Compatible Printer Driver (Microsoft Knowledge Base) Article ID: Q177512http:/support.microsoft.com/support/kb/articles/Q177/5/12.asp
