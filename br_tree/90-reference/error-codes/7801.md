**Error:** 7801

**Description:** Unreconciled keyword name

**Details:** Use of an invalid specification in the SORT control file. Run PRESORT and check the SORT specifications for syntax errors; use no keywords other than FILE, RECORD, SUM, MASK, ALTS or! (for a comment). If PRESORT is not used, check the punctuation in the program that creates the sort control file.
