**Error:** 2198

**Description:** No Default Printer.

**Details:** This error occurs when {PRN}:/DEFAULT is specified, but there is printer set as the windows default printer. Sometimes, usually in a case of corruption or viruses (i.e. Nimda) Windows 98 does not load the printer drivers at all, you can simply reboot the computer and the process will work.
