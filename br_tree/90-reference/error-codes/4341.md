**Error:** 4341

**Description:** Novell error 141

**Details:** As of BR! version `4.16` This error no longer exists.  It was replaced by a combination of Error Number 4320 used in conjunction with the system variables SYSERR and SYSERR$.Operation partially fails due to file/s in use. A station has attempted to delete, rename, or set file attributes using an ambiguous file name. The operation was only partially successful because one or more of the files specified was in use by another station.
