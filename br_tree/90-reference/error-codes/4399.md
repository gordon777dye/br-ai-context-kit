**Error:** 4399

**Description:** Windows file not shareable or (as of 4.3) File name is not a UNC path

**Details:** - Windows file is not shareable.
- OS drive lost map, while BR drive map is maintained.
- Drive statement contains a "." #
- re-map your drive in your OS
- Change the "." to something like "\\Server\Resource" or "C:\" (without the quotes)
