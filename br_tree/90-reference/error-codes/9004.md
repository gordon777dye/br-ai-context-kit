**Error:** 9004

**Description:** `FOR/NEXT` `stack` full

**Details:** The FOR/NEXT stack is full. Reduce the number of FOR/NEXT variables used in the program or increase the size of the `FORSTACK` setting in the `BRCONFIG.SYS` file (keep in mind that FOR/NEXT stack space comes from user memory).
