**Error:** 6245

**Description:** Invalid or unsupported `ESC sequence` under `Native Windows Printing`

**Details:** Invalid or unsupported ESC sequence under Native Windows Printing
Note that such escape sequences are not processed until the end of each print line. So lines that end with a semicolon typically will not generate this error. 

Don't forget that CLOSE #nn,FREE: will remove the currently spooled file.
