**Error:** 2108

**Description:** `Program` is active

**Details:** Program is active. Attempt at using `Run` to restart an interrupted program, or to `Save`/`Replace` a running program. Use the `End` statement or `Go` `command`, then issue the Run command to resume execution again. If you wish to save a changed program, use `List` in the form LIST  >program.brs, then load the file from source after the program has ended.
