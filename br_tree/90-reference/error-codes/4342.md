**Error:** 4342

**Description:** Novell error 142

**Details:** As of BR! version `4.16` This error no longer exists.  It was replaced by a combination of Error Number 4320 used in conjunction with the system variables SYSERR and SYSERR$.Operation fails due to file/s in use. A station has attempted to delete, rename, or set attributes but the operation failed because the files specified by the file name are in use by another station
