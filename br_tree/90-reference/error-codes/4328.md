**Error:** 4328

**Description:** Novell error 128 or Unix / Linux error 28

**Details:** As of BR! version `4.16` This error no longer exists.  It was replaced by a combination of Error Number 4320 used in conjunction with the system variables SYSERR and SYSERR$.Novell: File locked, an attempt has been made to open or create a file that is already open with a personal file attribute status.Unix / LinuxNo space left on device. Attempt to write to an ordinary file when there was no space left on the device
