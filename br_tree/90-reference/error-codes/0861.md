**Error:** 0861

**Description:** Invalid `format specification`

**Details:** - The format specification in a `field` definition `parameter` is either invalid or missing.
- If working with grids you may have omitted the `LIST` or `GRID` keyword from the fields spec. #Check to see that the format specification is present and that it is `C`, `N`, `V`, `G`, `L`, or `PIC`. Also make sure that there is a blank space between the format specification and the field length. Specify `Option (Config) Option` 45 to use old format
- add the List or Grid keyword
