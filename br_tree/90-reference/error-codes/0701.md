**Error:** 0701

**Description:** Page overflow (See also: `PAGEOFLOW` or `BRConfig.sys#PAGEOFLOW`)

**Details:** The number of printed lines equals or exceeds the number specified with the `PAGEOFLOW` parameter of the `OPEN DISPLAY` statement; the default number is 60 lines. Issue a PRINT #X:NEWPAGE to reset the PAGEOFLOW count.
