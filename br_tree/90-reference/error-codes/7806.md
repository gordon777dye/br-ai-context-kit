**Error:** 7806

**Description:** Syntax error in ALTS specification

**Details:** A syntax error occurred in the ALTS specification. Run PRESORT and check the ALTS specification in the sort control file for typing errors,missing punctuation, etc. If PRESORT is not used, check the punctuation in the program that creates the sort control file.
