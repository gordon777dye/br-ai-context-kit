**Error:** 4323

**Description:** Unix / Linux error 23

**Details:** As of BR! version `4.16` This error no longer exists.  It was replaced by a combination of Error Number 4320 used in conjunction with the system variables SYSERR and SYSERR$.File table overflow. The system file table is full; additional opens must temporarily be denied.  (Not in windows, see cause 2)
- Bad File Name, Path Name, Directory Name, etc.
- combined Path and File Name exceed 35 characters
- Bad pathname in library declaration, error happened when first library was called. # Fix the Name
- Shorten either the path or the filename.
- Fix the library declaration pathPREVMFiles in System.ini does not helpNote: (Windows) Not related to the number of open files.
