**Error:** 4212

**Description:** DEPRECATED ERROR, as of 4.30: `Operating System Error` 12 (Invalid access code)

**Details:** This error would normally occur in Business Rules! only if memory has been corrupted. Invalid access code. Try reloading from source. If this doesn't work, contact your dealer or `BRC` for assistance.
