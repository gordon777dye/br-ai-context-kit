**Error:** 6298

**Description:** Printer error

**Details:** - Printer is not ready (on-line), is out of paper, or is otherwise not functioning.
- you are using an older version of Business Rules! with a newer program that has /SELECT on the open #255 line. # Fix the printer. If the status line says ERROR, type GO to continue the program after the printer has been fixed.
- list the offending line and change the /SELECT into a /10 to tell the program to use the default printer instead of asking which printer to use.
