**Error:** 0309

**Description:** Illegal nesting of `Def`

**Details:** - The program illegally nests DEF statements; one `user-defined function` cannot be contained within another.
- possibly `line numbers` out of order #Eliminate nested Def statements.
- renumber the non consecutive line numbers
