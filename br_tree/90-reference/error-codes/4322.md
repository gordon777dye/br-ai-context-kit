**Error:** 4322

**Description:** Unix / Linux error 22 (Invalid argument)

**Details:** As of `BR!` version `4.16` This error no longer exists.  It was replaced by a combination of Error Number `4320` used in conjunction with the system variables `SYSERR` and `SYSERR$`.This error would normally occur in Business Rules! only if memory has been corrupted. Invalid argument. Try reloading from `source`. If this doesn't work, contact your dealer or `BRC` for assistance
