**Error:** 4211

**Description:** DEPRECATED ERROR, as of 4.30: `Operating System Error` 11 (Invalid format)

**Details:** This error would normally occur in Business Rules! only if memory has been corrupted. Invalid format. Try reloading from source. If this doesn't work, contact your dealer or `BRC` for assistance
