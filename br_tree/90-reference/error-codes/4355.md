**Error:** 4355

**Description:** Novell error 155

**Details:** As of BR! version `4.16` This error no longer exists.  It was replaced by a combination of Error Number 4320 used in conjunction with the system variables SYSERR and SYSERR$.Bad directory handle. Attempt to access a directory that does not exist.
