**Error:** 4312

**Description:** Unix / Linux error 12

**Details:** As of BR! version `4.16` This error no longer exists.  It was replaced by a combination of Error Number 4300 used in conjunction with the system variables SYSERR and SYSERR$.Not enough space, a program is requesting more space than the system is able to supply.
