**Error:** 0603

**Description:** `Translate` table not found

**Details:** Translate table, specified after TRANSLATE<nowiki>=</nowiki> in an `Open Display` `statement` cannot be located. Correct errors in the file name or path for the translate table; be sure that drive and path are valid; check file permissions.
