**Error:** 4308

**Description:** DEPRECATED ERROR, as of 4.30: Unix / Linux error 8

**Details:** As of BR! version `4.16` This error no longer exists.  It was replaced by a combination of Error Number 4300 used in conjunction with the system variables `SYSERR` and `SYSERR$`.This error would normally occur in Business Rules only if memory has been corrupted. This is an execute format error. Try reloading from source. If this doesn't work, contact your dealer or `BRC` for assistance.
