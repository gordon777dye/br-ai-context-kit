**Error:** 7600

**Description:** Illegal or missing key start position

**Details:** - illegal starting position specified for the key
- no starting position or an was specified for the key.
- `rln` of file is smaller than the positions specified #Make sure that the key starting position and length are specified. 
- The starting position must be a number greater than zero.
- The starting position nor the combined key starting position and key length may not exceed the record length.
