**Error:** 9307

**Description:** Illegal `On Error` `statement`

**Details:** - Use of an illegal On Errorstatement.
- `Edit` `command` used but no `Editor` has been selected via a `Config` Editor statement nor `BRConfig.sys` configuration. # Save the program as source and reload it as source. If the error persists, call your dealer or `BRC` for help.
- Use Config Editor command or edit your BRConfig.sys to include the Editor and restart BR! and try the Edit command again.
