**Error:** 9304

**Description:** System error on user defined function

**Details:** - User-defined function error.
- Or perhaps you added a FNwhatever to a running program # Save the program as source and reload it as source. If the error persists, call your dealer or `BRC` for help.
- `END` the program add the LIBRARY "Library_Filename": FNwhatever line to the program and then it should work.
