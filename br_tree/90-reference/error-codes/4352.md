**Error:** 4352

**Description:** Novell error 152

**Details:** As of BR! version `4.16` This error no longer exists.  It was replaced by a combination of Error Number 4320 used in conjunction with the system variables SYSERR and SYSERR$.Volume does not exist. The Network operating system is trying to access a volume and cannot find the volume in the system definitions files
