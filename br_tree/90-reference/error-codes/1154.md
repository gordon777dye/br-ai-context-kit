**Error:** 1154

**Description:** Unexpected formatting applied during `Edit`

**Details:** - During an edit an extra space was added after a !: (`line continuation`)
- During an edit a line which was too long was encountered.
- During an edit a line continuation error was encountered, in one such case the preceding line did not have a !: where it should have been.
- During an edit a `comment` was found on a line with no line number
- During an edit an `!` (exclamation point for a comment) was immediately following a line number without a space in between.
- use of a 31 character variable name.  (30 is the maximum.) If none of the following apply and you are using EDI, try reloading the editor's source file into BR! as `source`.
- Repair offending line in Source code and re-import that code into BR!
- Repair offending line in Source code and re-import that code into BR!
- Repair offending line in Source code (by adding a !: in the appropriate place) and re-import that code into BR!
- Add a `line number` or add a `line continuation` to the previous line
- Add the missing space between the line number and the !
- To find this error one may
 clear all
 load path\program.br.brs,so
 list -99999
 the line after the last one listed was the problem.  (or at least the first instance of it)
