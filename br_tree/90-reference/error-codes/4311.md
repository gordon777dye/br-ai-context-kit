**Error:** 4311

**Description:** Unix / Linux error 11

**Details:** As of BR! version `4.16` This error no longer exists.  It was replaced by a combination of Error Number 4300 used in conjunction with the system variables SYSERR and SYSERR$.No more processes. The system process table is full or the user is not allowed to create any more processes
