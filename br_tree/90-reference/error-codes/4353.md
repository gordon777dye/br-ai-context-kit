**Error:** 4353

**Description:** Novell error 153

**Details:** As of BR! version `4.16` This error no longer exists.  It was replaced by a combination of Error Number 4320 used in conjunction with the system variables SYSERR and SYSERR$.Directory is full. The Network operating system ran out of entry space in a directory.
