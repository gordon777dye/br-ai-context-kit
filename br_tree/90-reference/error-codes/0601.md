**Error:** 0601

**Description:** `Key file` name invalid

**Details:** Key file name is missing or invalid.This error can also occur when `S/23-MODE` is ON and an `OPEN` `statement` tries to open a `program` file as an `internal` file. Specify a valid key file name.If S/23-MODE is ON, correct the NAME= parameter in the OPEN statement so that it refers to an internal file.
