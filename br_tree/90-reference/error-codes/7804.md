**Error:** 7804

**Description:** Syntax error in file specification

**Details:** A syntax error occurred in the FILE specification. Run PRESORT and check the FILE specification in the sort control file for typing errors, missing punctuation, etc. If PRESORT is not used, check the punctuation in the program that creates the sort control file.
