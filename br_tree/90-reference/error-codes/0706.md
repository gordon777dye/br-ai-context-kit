**Error:** 0706

**Description:** Illegal `file handle` no access to 8087 chip

**Details:** - Specification of a `file handle` outside the correct range
- An 8087 model of `Business Rules!` is not able to access a math co-processor chip
- Attempt to `Restore` an `Open Internal unopened` file handle. # Change it so that it is 1 through 199, or 300-999.
- Check to see that the 8087, 80287 or 80387 chip is installed properly on the machine and that all switches are set properly.
- Properly open the file before restoring it.
