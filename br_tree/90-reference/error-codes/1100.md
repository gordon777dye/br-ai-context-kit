**Error:** 1100

**Description:** Illegal multiple `statement` operations

**Details:** Illegal multi-statement operation.  BR! does not permit the use of `Data`, `Def`, `Dim`, `Exit`, `Form` or `Option (Statement) Option` statements on the same program `line number` as any other statement. Place the offending instruction on its own program line.
