**Error:** 0715

**Description:** Illegal sequence

**Details:** - Use of an illegal sequence for `REREAD`, `REWRITE`, or `DELETE`.
- A `SAME` positional parameter is used when no record has previously been referenced. # Make sure that REREAD, REWRITE, or DELETE follows a successful READ or REREAD, or make sure that REC<nowiki>=</nowiki> or KEY<nowiki>=</nowiki> are used.
- Make sure the record has been referenced prior to the use of the SAME positional parameter.
