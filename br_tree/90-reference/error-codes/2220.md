**Error:** 2220

**Description:** Missing or duplicate file specification

**Details:** - Occurrence of a missing file specification (i.e. LOAD )
- Occurence of a duplicate file specification (i.e. RUN  >DUP  >DUP), or a duplicate file. #Supply the missing file specification.
- Remove the duplicate file specification.
