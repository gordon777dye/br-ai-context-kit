**Error:** 0106

**Description:** `Array` size conflict

**Details:** One of two problems occurred:
- The array sizes in a `Mat` statement do not match or,
- Mat grouping (in a `Input`/ `Print`/ `RInput` Fields statement) was attempted with mismatched array sizes. # Make sure that the `dimension` size of the arrays in the Mat statement are the same.
- Make sure that the dimension size of the arrays in the Mat grouping are the same.
