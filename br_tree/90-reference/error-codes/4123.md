**Error:** 4123

**Description:** Incomplete record or corrupted B-Tree file.

**Details:** Incomplete record due to corrupted `B-tree` file.  It may be related to the method used for 31 `bit locking` if the index file exceeds the allowed size. Re-`Index` the file.
