**Error:** 4217

**Description:** DEPRECATED ERROR, as of 4.30: Operating System Error 17.

**Details:** Not the same device (or drive). New path and file names specified with the RENAME command must exist on the same drive or device as the original. Re-specify the RENAME command using the correct file specification. New file and path names specified with the RENAME command must exist on the same drive or device as the original.
