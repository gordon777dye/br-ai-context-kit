**Error:** 4203

**Description:** `Operating System Error` 3. Path does not exist or is not a directory

**Details:** One of the following situations has occurred:
- Bad `path` in a `library statement`
- Invalid path name. The specified path cannot be found; could not locate the directory as specified.
- `Index` file name and work directory name may be the same.
- Invalid `WORKPATH` specified in `BRConfig.sys`. # Correct the path; be sure that the specified directories exist on the disk in the specified drive and that the correct default directory is in use.
- Be sure that the work directory is separate from the INDEX directory.Correct the WORKPATH specified in the BRConfig.sys file
- See first resolution.
- See first resolution.
