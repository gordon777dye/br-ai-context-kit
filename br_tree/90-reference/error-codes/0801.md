**Error:** 0801

**Description:** Invalid syntax for `Form` `Variable`

**Details:** - Invalid syntax for a Form `string` variable. Unable to compile A$ in `Cnvrt$`(A$,X) or PRINT, USING A$.
- Attempt to check the `KLn` of a `relative` (not-keyed) file.
- Attempt to `CForm$` a form statement with a `BH` field that is too large. #Change the contents of A$ to valid format specifications. In PRINT USING A$, the FORM string A$ must begin with the word *Form* followed by one or more blank spaces.
- Change the reference to an indexed file or remove the KLn
- Correct the form statement
