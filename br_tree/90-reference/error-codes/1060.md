**Error:** 1060

**Description:** Source statement or `Config` `command` too long

**Details:** - A source statement or a CONFIG command is too long.
- Listing of a Program Fails # Shorten the statement to 800 characters or less, or shorten the Config command to 128 characters or less.
- If, for example, you OPEN #0: "Cols<nowiki>=</nowiki>100" and load an existing program with very long lines and attempt to list it you will get this error.
As of 10/08/2006 the BR! Editor is not yet ready for this, please set the columns back to 80 and try again. Please note that changing the number of rows does not cause this error.
