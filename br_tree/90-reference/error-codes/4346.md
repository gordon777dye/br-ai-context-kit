**Error:** 4346

**Description:** `Novell` error 146

**Details:** As of BR! version `4.16` This error no longer exists.  It was replaced by a combination of Error Number `4320` used in conjunction with the system variables `SYSERR` and `SYSERR$`. `Rename` operation fails (other names already exist). A station has attempted to rename one or more files, but the operation failed because the file names specified already exist.
