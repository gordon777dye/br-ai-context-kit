**Error:** 4201

**Description:** DEPRECATED ERROR, as of 4.30: `Operating System Error` 1

**Details:** This error would normally occur in Business Rules only if memory has been corrupted. Invalid function number. Try reloading from source. If this does not work, contact your dealer or `BRC` for assistance.
