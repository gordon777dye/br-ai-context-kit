**Error:** 6242

**Description:** Bad parameterized printer substitutions

**Details:** The parameters in the substitution value do not match the values in the prototype. This occurs during printing when specifying printer substitutions such as:  
 PRINTER NWP [POS(ROW,COL)], "\E&aROWrCOLC"  
In this case the program must provide exactly two parameter values. A variety of mismatch conditions can cause this error.
