A **Debug Release** is a release not meant for end users, but for developers doing `Beta` testing for `BRC`.  There are two release types: Debug Release, and `Native Release` is the other one.

It has several key differences:

*Slower
*Creates `br-electric-fence-leaks.txt` files
*Uploads Error reports via FTP to BRC
