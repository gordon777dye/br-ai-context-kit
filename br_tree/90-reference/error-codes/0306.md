**Error:** 0306

**Description:** `FnEnd` not matched with `Def` in program structure

**Details:** An FnEnd occurred without a preceding Def `statement` or in a programming structure such as a `For`/`Next` loop or a `GoSub`. Match the FnEnd with a Def statement; remove the `user-defined function` from the programming structure.
