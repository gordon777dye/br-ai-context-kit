**Error:** 4222

**Description:** DEPRECATED ERROR, as of 4.30: `Operating System Error` 22 (Unknown command)

**Details:** This error would normally occur in Business Rules only if memory has been corrupted. Try reloading from source. If this doesn't work, contact your dealer or `BRC` for assistance
