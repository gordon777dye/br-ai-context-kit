**Error:** 4324

**Description:** Unix / Linux error 24

**Details:** As of BR! version `4.16` This error no longer exists.  It was replaced by a combination of Error Number `4320` used in conjunction with the system variables `SYSERR` and `SYSERR$`. Too many open files. No process can have more than 20 file descriptors open at one time
