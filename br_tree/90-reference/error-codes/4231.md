**Error:** 4231

**Description:** DEPRECATED ERROR, as of 4.30: `Operating System Error` 31

**Details:** Operating System has detected an unspecified problem with a disk drive or other device. You may be attempting to write to an unformatted diskette. Try another diskette. Check to see if you are using a low-density drive to access a diskette, which has been written to, by a high-density drive.
