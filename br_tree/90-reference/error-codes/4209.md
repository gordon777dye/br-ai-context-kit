**Error:** 4209

**Description:** DEPRECATED ERROR, as of 4.30: `Operating System Error` 9 (Invalid memory block address)

**Details:** This error would normally occur in Business Rules only if memory has been corrupted. It is an invalid memory block address. Try reloading from source. If this doesn't work, contact your dealer or `BRC` for assistance.
