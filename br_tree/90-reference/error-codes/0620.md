**Error:** 0620

**Description:** DEPRECATED ERROR: Duplicate OPEN for standard printer output

**Details:** Two attempts were made to establish a destination for standard printer output (#255). A common form of this error is trying to use output re-direction twice on the same command:E.g., RUN  >LST  >LST. Remove one of the specifications for standard printer output.
