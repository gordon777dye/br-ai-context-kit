**Error:** 0413

**Description:** Window Not On Top

**Details:** A Keyboard message was sent to a task which is different from the one that currently owns the keyboard.This error is defined in support of the command `Msg` - controlling the keyboard through Windows Messaging. (??Try setting the focus to the proper window before sending the message??)
