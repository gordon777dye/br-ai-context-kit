A **Debugger** is an `third party utility` that communicates with `BR32.exe` to enhance your coding experience.

Currently `MyEdit BR` was the first developed `editor` for use with `Business Rules!`. See the documentation [http://www.mills-enterprise.ca/download/products/myeditbrv3help.pdf on it's website]
