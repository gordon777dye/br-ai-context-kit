**Error:** 4280

**Description:** DEPRECATED ERROR, as of 4.30: `Operating System Error` 80

**Details:** File exists, cannot create new file. Example: Tried to overwrite memory swap file but couldn't. It is likely that memory is corrupted. Contact your dealer or `BRC` for assistance.
