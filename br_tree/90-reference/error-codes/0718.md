**Error:** 0718

**Description:** `Key length` (KLn) conflict

**Details:** I/O statement specifies a key length that does not match the key length specified in the `Open` `statement`. 

Some possibilities:
- The length of the `SEARCH` specification is greater than the key length.
- The key length of the specified key does not match the key length for the file.
- The length of the <nowiki>LINK=</nowiki> string does not match the key length specified in OPEN statement ( releases `3.60` +) Correct the key length error. For problems described above:
- Make sure the SEARCH specification is less than or equal to the key length
- Match the key lengths by changing either the error-causing `READ` statement or the OPEN statement.
- Same as 2 (above)
