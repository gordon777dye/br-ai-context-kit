**Error:** 0108

**Description:** `Maximum String Length` (`32,767`) Exceeded

**Details:** A string functions yields a result that exceeds the maximum length of a string (32,767 bytes), such as <nowiki>RPT$("=", 36000)</nowiki>. Rearrange your logic so that you may manage multiple pieces of the large string.
