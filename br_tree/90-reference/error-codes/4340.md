**Error:** 4340

**Description:** (`4.2`+) An HTTP error has occurred / (`4.16`-) Novell error 140

**Details:** In BR! version `4.20`, This error code was re-introduced to be used in conjunction with the `SysErr` and `SysErr$`. It indicates that an error occurred in the underlying operating system. Reference SysErr$ for a description of the actual error. 

As of BR! version 4.16 This Novell error no longer exists. Use the SysErr & Syserr$ to troubleshoot the errors.   The HTTP interface in Business Rules uses the CURL library, and these errors should be documented in the Curl Site.
