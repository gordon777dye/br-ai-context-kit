**Error:** 0760

**Description:** `Picture` does not exist

**Details:** If a referenced graphic does not exist BR used to display a grey blob, but not error.  With the latest `4.2`  releases this no longer happens, an error 760 is generated for a non-existent graphic. (as of `March 20`th, `2009`) May be generated on the line that closes the printer and not the line that attempts to print the picture. Correct the picture file location/path.
