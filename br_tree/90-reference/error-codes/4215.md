**Error:** 4215

**Description:** `Operating System Error` 15.  Invalid drive reference

**Details:** Some possibilities:
- Illegal `drive` reference.
- Invalid drive in # Change the drive reference to a drive that exists.
- Correct the `BRConfig.sys` file's `WorkPath` specification.
