**Error:** 4347

**Description:** Novell error 147

**Details:** As of BR! version `4.16` This error no longer exists.  It was replaced by a combination of Error Number 4320 used in conjunction with the system variables SYSERR and SYSERR$.No read privileges. A station has attempted to read a file for which it has no read privileges
