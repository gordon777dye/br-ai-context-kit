**Error:** 2003

**Description:** Illegal line number sequence

**Details:** Use of an illegal line number sequence. In the following example, a LIST command attempts to list lines in reverse order; this line would cause this error: LIST 500 100 Line numbers must be referenced in sequential order; change any references that are incorrect.
