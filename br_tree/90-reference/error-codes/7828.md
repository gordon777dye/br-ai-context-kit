**Error:** 7828

**Description:** SORT specs out of order

**Details:** SORT specifications are out of order or there may be two or more FILE specifications in the same sort control group. In the program that creates the sort control file, delete the extra FILE specification and/or reorganize the SORT specifications in the following order: ! (for a comment), FILE, ALTS, SUM, RECORD and MASK. MASK must always be last
