**Error:** 2280

**Description:** `Library` not executable

**Details:** A `Load` `Resident` `command` was issued for a library `program` that is not executable. The Load operation was not performed. Fix the bug in the library `program`, execute `Clear Resident` and run your program again.
If you get this in a screenio screen, then load and replace it (lo screenio\screenname, rep) to find the real location of the error.
