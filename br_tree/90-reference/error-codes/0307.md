**Error:** 0307

**Description:** Inactive user defined function

**Details:** Attempt to assign a value to a user-defined function that is not active. Correct the program. The most likely cause is a `GOTO` to the middle of a user-defined function. A FN Variable cannot be assigned unless the user-defined function it names has been called and the `FNEND` has not yet been encountered.
