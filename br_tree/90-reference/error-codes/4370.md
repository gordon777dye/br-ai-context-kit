**Error:** 4370

**Description:** (deleted) same as `4148`

**Details:** As of BR! version `4.16` This error no longer exists.  It was replaced by a combination of Error Number 4320 used in conjunction with the system variables SYSERR and SYSERR$.Starting with version 3.50, error code 4148 (file sharing violation) is returned for these situations instead.
