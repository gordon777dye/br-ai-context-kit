**Error:** 0879

**Description:** Column Data Type

**Details:** Attempting to use both `strings` and `numbers` in a G, FMT, PIC column. In a G spec column, the column becomes either a string column or number column based on the first element. BR now stores data internally in BR internal format rather than storing numeric data as strings.
