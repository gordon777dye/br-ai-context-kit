**Error:** 4301

**Description:** DEPRECATED ERROR, as of 4.30: Unix / Linux error 1

**Details:** As of BR! version `4.16` This error no longer exists.  It was replaced by a combination of Error Number 4300 used in conjunction with the system variables SYSERR and SYSERR$.Not owner. Unauthorized attempt to modify file. Must be file owner or super-user.
