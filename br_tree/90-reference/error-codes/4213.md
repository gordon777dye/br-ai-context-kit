**Error:** 4213

**Description:** DEPRECATED ERROR, as of 4.30: `Operating System Error` 13 (Invalid data)

**Details:** This error would normally occur in Business Rules only if memory has been corrupted. Invalid data. Try reloading from source. If this doesn't work, contact your dealer or `BRC` for assistance.
