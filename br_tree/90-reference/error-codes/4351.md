**Error:** 4351

**Description:** Novell error 151

**Details:** As of BR! version `4.16` This error no longer exists.  It was replaced by a combination of Error Number 4320 used in conjunction with the system variables SYSERR and SYSERR$.No disk space for spooled file. The Network operating system has determined that the Network disk is out of space for spooled files
