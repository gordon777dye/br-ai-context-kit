**Error:** 2090

**Description:** not enough compilation storage

**Details:** - Insufficient storage to compile or store a program line.
- Tried to Load a program compiled in 3.91 in 3.32 # Eliminate some arrays and/or make them smaller or delete program lines to make the program smaller. Also try saving the program and loading it again
- Save program source in 3.91 and load source in 3.32
