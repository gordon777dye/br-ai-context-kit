**Error:** 0410

**Description:** Invalid `internal function` parameter

**Details:** - An invalid parameter is used for a `HEX$` internal function.
- An invalid parameter is used for a `PIC$` internal function.
- An invalid parameter is used for a `LPAD$` internal function.
- An invalid parameter is used for a `LTRM$` internal function.
- An invalid parameter is used for a `RPAD$` internal function.
- An invalid parameter is used for a `RTRM$` internal function.
- An invalid parameter is used for a `TRIM$` internal function. # HEX$: The `hexadecimal` character must contain an even number of characters. All characters for HEX$ must be either digits 0 to 9 or uppercase letter A through F.
- PIC$: Argument must be one character long.
- LPAD$: The "char" parameter may be no longer than one character.
- LTRM$: The second parameter may be no longer than one character.
- RPAD$: See LPAD$ (above)
- RTRM$: See LTRM$ (above)
- TRIM$: The second parameter may be no longer than one character.
