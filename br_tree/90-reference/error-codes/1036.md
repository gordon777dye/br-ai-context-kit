**Error:** 1036

**Description:** Illegal `string` data

**Details:** - String data occurs in a `numeric` expression.
- some_string$<nowiki>=</nowiki>some_number #String expressions may contain only string data; numeric expressions may contain only numeric data. Fix the error-causing expression.
- some_string$<nowiki>=</nowiki>`str$`(some_number)
