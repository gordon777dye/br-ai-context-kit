In `Business Rules!` **Open (OPE)** is used only in `statements`.

All BR **Open** statements use the format:

 00150   Open #channel-no: "open-string", file-type, out-in [, index-type ]

Here are the different types of Open statements ( based on file-type ).

* `Open Communications`
* `Open Display`
* `Open External`
* `Open Internal`
* `Open Window`
* `Name=Save:`
* `Name=Open:`
* `Linked`
* `Opening a Print File`
