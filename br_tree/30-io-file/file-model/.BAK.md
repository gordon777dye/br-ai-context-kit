**.bak** (`wikipedia:.bak|wikipedia`) is the `wikipedia:file extension|file extension` for a `Business Rules!` **program backup file**.  

When a programmer is about to overwrite an existing program file (`.br` (for example, when the user `replace|replaces` the program he or she is working on), by default BR! will first make a copy of the existing file, replacing the extension with .bak.  

To change this default behavior you may implement `Option (Config)|Option` 26.
