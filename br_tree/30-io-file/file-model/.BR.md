**.br** is the default `wikipedia:file extension|file extension` for a `Business Rules!` **program file**.  However for legacy purposes some program development groups utilize a **.wb** alternative.  One may add `Option (Config)|Option` 24 to your `BRConfig.sys` to change the default extension from .br to .wb.

Program Files are a semi-compiled version of `source` code.

The three states that a program may exist in are:

*.br/.wb - a regular program file
*.brs/.wbs - `source code`
*.bro/.wbo - `object code`

Additionally a program may be a combination of .bro and .br.  In these cases the file is still generally given the .br extension.

From a `ready prompt` you can `load` or `run` a program file.

===File Association===
To associate .br or .wb files with your Business Rules! so that you can edit or run them by double clicking or starting them in windows see the `Double Click Project`.
